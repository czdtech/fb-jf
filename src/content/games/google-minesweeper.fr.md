### Présentation du jeu

Minesweeper affiche des nombres indiquant combien de mines touchent chaque case. En t’appuyant sur ces indices, tu dois poser des drapeaux sur les mines et ouvrir toutes les cases sûres, avec pour objectif de vider le plateau sans aucune explosion. La difficulté ajuste la taille de la grille et la quantité de mines.

### Comment jouer

- Clic gauche pour ouvrir une case ; clic droit pour placer/retirer un drapeau. Certains hôtes permettent un « chord » : un clic spécial autour d’un nombre quand le bon nombre de drapeaux l’entoure.
- Utilise les inférences de base (paires 1‑1, motifs 1‑2, etc.) pour agrandir tes zones sûres.
- Sur les grandes grilles, travaille par coins et bords pour limiter les embranchements possibles.

### Commandes

- Souris / tactile ; les raccourcis pour le chord et les ouvertures rapides dépendent de la version.

### Astuces

- Adopte une convention de marquage stable pour éviter les drapeaux mal placés.
- Quand tu dois deviner, choisis des cases qui, en cas d’erreur, révèleront le plus d’informations.
- Les bords contraignent souvent les patterns : scanne régulièrement les bordures pour repérer des coups sûrs.

### Foire aux questions (FAQ)

- Q : Comment éviter les situations où je dois deviner ?
  
  R : Utilise les motifs standard et les vérifications de parité ; malgré tout, certains cas resteront des 50/50 inévitables.

- Q : Que signifie le nombre sur une case ?
  
  R : Il correspond au nombre de mines présentes dans les huit cases adjacentes.

- Q : La difficulté change‑t‑elle seulement la taille de la grille ?
  
  R : Non, elle modifie à la fois les dimensions du plateau et la densité de mines.
