### Présentation détaillée

Kings and Queens reprend les bases du match‑3 et les associe à une progression de royaume : tu complètes les niveaux pour gagner des ressources et restaurer un décor royal (château, jardins, etc.). Chaque chapitre introduit de nouveaux bloqueurs (chaînes, dalles de marbre, brouillard) et des objectifs de scène (collecter des couronnes, casser des sceaux). Tu dépenses les étoiles gagnées pour rénover les pièces, ce qui débloque des boosters pour les tableaux plus difficiles.

### Stratégies de jeu

1.  **Contrôle du plateau** : nettoie d'abord les bloqueurs au centre pour créer de l'espace et faciliter les combos.
2.  **Timing des boosters** : garde marteaux / fusées pour les zones clef en fin de niveau.  
3.  **Planification des combos** : fusionne les power‑ups (bombe + ligne, bombes de couleur…) juste avant les objectifs difficiles.  
4.  **Gestion des étoiles** : dépense‑les dans les boosters qui t'aident vraiment sur les niveaux où tu bloques.
5.  **Compter les coups** : garde en tête le nombre de bloqueurs restants vs les coups disponibles, et joue pour l'objectif plutôt que pour le score brut.

### Commandes

- Glisse pour échanger ; tape sur les boosters pour les utiliser ; menu de scène pour rénover avec les étoiles.

### Foire aux questions (FAQ)

- **Q : Je suis bloqué sur un niveau avec des dalles de marbre, une astuce ?**
  R : Utilise les bombes près des dalles et enchaîne les balayages de ligne pour fissurer plusieurs couches.

- **Q : Dois‑je garder mes boosters en réserve ?**
  R : Garde‑en quelques‑uns, mais dépense‑les pour franchir les niveaux difficiles ; la progression compte plus que l'accumulation.

- **Q : Les rénovations affectent‑elles le gameplay ?**
  R : Elles sont principalement cosmétiques ; certaines débloquent des emplacements de boosters ou des récompenses quotidiennes.
