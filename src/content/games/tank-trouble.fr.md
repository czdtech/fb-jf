### Presentation du jeu

Tank Trouble est un jeu de bataille de chars multijoueur ou tu controles un char dans une arene labyrinthique, tirant sur tes adversaires tout en naviguant entre les murs et obstacles pour etre le dernier char debout. Le jeu est populaire pour ses combats rapides et son gameplay de positionnement strategique.

### Guide de jeu

1.  **Utilise les murs :** Les balles ricochent sur les murs ; apprends a utiliser les rebonds pour des attaques indirectes sur tes adversaires.
2.  **Strategie de mouvement :** Continue de bouger pour eviter de devenir une cible immobile, mais controle aussi tes angles de tir.
3.  **Avantage du terrain :** Utilise le terrain du labyrinthe pour bloquer les trajectoires de tir ennemies.
4.  **Tir predictif :** Anticipe les directions de mouvement de tes adversaires et vise en avance.
5.  **Collecte les bonus :** Ramasse les bonus speciaux dans l'arene pour obtenir des avantages.

### Commandes

*   **Controle de mouvement :** Utilise les fleches ou WASD pour controler le mouvement et la rotation du char.
*   **Tir :** Utilise les touches designees (generalement Espace ou touches specifiques) pour tirer.
*   **Mode multijoueur :** Supporte le multijoueur local avec differentes combinaisons de touches pour chaque joueur.

### Foire aux questions

1.  **Q : Comment les balles ricochent-elles ?**
    **R :** Les balles rebondissent sur les murs selon les regles de physique ; tu peux utiliser ce mecanisme pour des attaques indirectes.
2.  **Q : Comment eviter de me faire toucher ?**
    **R :** Continue de bouger, utilise les murs comme couverture et essaie de predire les trajectoires de tir de tes adversaires.
3.  **Q : Combien de joueurs peuvent jouer simultanement ?**
    **R :** Generalement 2-3 joueurs en batailles multijoueur locales.
4.  **Q : Y a-t-il une limite de temps ?**
    **R :** Certains modes peuvent avoir des limites de temps, mais l'objectif principal est de vaincre tous les adversaires.
