### Présentation du jeu

Cube Tower reprend le principe du tower defense avec une esthétique en cubes.
Des vagues d’ennemis suivent un chemin défini et tu dois placer des cubes de
 défense le long de ce chemin pour les éliminer avant qu’ils n’atteignent la
fin.

Chaque type de cube a ses propres caractéristiques (portée, dégâts, vitesse de
 tir, effets spéciaux). Tu peux les améliorer avec les ressources gagnées.

### Guide de jeu

1.  **Apprends le chemin des ennemis**  
    Repère les zones où les monstres restent le plus longtemps (virages,
    segments lents) pour y placer tes cubes les plus puissants.
2.  **Varie les types de cubes**  
    Combine dégâts rapides, ralentissements et gros coups à fort impact pour un
    meilleur contrôle des vagues.
3.  **Améliore plutôt que spammer**  
    Il est souvent plus efficace d’améliorer un bon cube bien placé que de
    remplir la carte de cubes de base.
4.  **Garde des ressources pour les dernières vagues**  
    Les vagues finales sont souvent les plus difficiles : prévois un budget
    pour des upgrades de dernière minute.
5.  **Teste différentes stratégies**  
    Certaines cartes favorisent les ralentissements, d’autres les gros dégâts en
    zone – adapte‑toi.

### Commandes

- **Placer un cube** : clique/tape sur une case valide près du chemin, puis
  choisis le type de cube.  
- **Améliorer / Vendre** : sélectionne un cube déjà posé pour ouvrir le menu
  d’upgrade ou de vente.  
- **Contrôles de vague** : lance la vague suivante ou accélère le temps via les
  boutons de l’interface.
