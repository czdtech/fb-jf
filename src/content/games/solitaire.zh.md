### 游戏介绍

《Solitaire》是最经典的单人纸牌接龙玩法之一。玩家需要根据花色与点数规则在桌面上移动纸牌，逐步把所有牌整理好并移入四个基础牌堆中。规则易学，但要想稳定获胜仍然需要一定的规划和策略，是一款经久不衰的益智小游戏。

### 游玩策略

1.  **优先翻开暗牌：** 尽量优先执行能翻开桌面暗牌的移动，这会大幅增加可选操作，提高解局机会。
2.  **聪明使用空列：** 空列只能放置 K 或以 K 开头的整段序列，是非常宝贵的临时缓冲空间，要慎重使用。
3.  **提前规划多步移动：** 许多局面需要“先挪开再搬回”的多步组合，动牌前先想好之后几步，避免把关键牌堵死。
4.  **基础牌堆不要一味往上堆：** 虽然把牌尽快移到基础堆很诱人，但有时你还需要这些牌在桌面上辅助其他移动，时机拿捏很重要。
5.  **善用废牌堆：** 通过合理利用废牌堆中的牌来完成关键移动，不要忽视从废牌回到桌面的可能性。

### 操作说明

*   **移动纸牌：** 点击并拖拽纸牌到合法位置，例如按“点数递减 + 红黑交错”的规则叠放到桌面列，或移入符合顺序的基础堆。
*   **翻牌：** 点击桌面列顶部的暗牌即可将其翻成明牌。
*   **自动移动：** 在多数数字版本中，双击一张可以直接移入基础堆的牌，系统会自动完成移动。
*   **悔棋：** 某些版本提供撤销功能，可回退一步操作，用于修正误点或重新思考。

### 常见问题解答（FAQ）

1.  **问：什么是基础牌堆？**  


**A**: Usually based on completion time, number of moves, and number of hints used.

- **Q: How does the scoring system work?**

**A**: Not all card layouts have solutions; when encountering unsolvable situations, you can start a new game.

- **Q: What if the game is unsolvable?**

**A**: Only card sequences arranged in descending order with alternating suits can be moved together.

- **Q: How do I move multiple cards?**

**A**: Foundation piles are four empty spaces that need to be filled with cards by suit from A to K in order.
