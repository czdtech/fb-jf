*   **[FR]** **Move**: Use left/right arrow keys or A/D keys to control horizontal character movement.
*   **[FR]** **Jump**: Use the up arrow key or W key to jump; some levels may support double or multiple jumps.
*   **[FR]** **Slide/Crouch**: Use the down arrow key or S key to slide or crouch, passing through low passages or dodging obstacles.
*   **[FR]** **Wall Jump**: When the character touches a wall, pressing the jump key allows for a wall jump; continuous wall jumps can be used to climb upwards.
