### Detailed Game Introduction

Aqua Blocks is a relaxing block‑placement game. Place given shapes to fill lines or regions, make space with multi‑line clears, and avoid creating sealed cavities.

### Gameplay Strategy & Walkthrough

1.  **[FR]** Keep edges tidy to prevent stray single cells.
2.  **[FR]** Build for Tetris‑like clears across multiple lines.
3.  **[FR]** Don’t over‑commit to one corner.
4.  **[FR]** Save emergency slots for awkward shapes.
5.  **[FR]** Use previews (if any) to plan sequences.

### Controls Guide

- Drag shapes onto the grid; rotate if supported.
- Undo/Reset via UI.
