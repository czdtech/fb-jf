### 详细游戏介绍

《Temple Run 2》延续了前作“亡命逃亡”的核心设定，你从被诅咒的神庙中夺走神像后，必须沿着蜿蜒曲折的道路一路狂奔。途中会遇到断桥、火陷阱、旋转木桩、矿车轨道等各种机关，还要顺手收集金币与宝石，用于解锁新角色和升级能力。

### 游玩策略与跑酷指引

1.  **视线放远：** 不要只盯着角色脚下，把视线略微擡高可以让你更早看到转弯、缺口和障碍。
2.  **熟悉操作手势：** 左右转弯、跳跃、滑铲、左右倾斜（重力感应）等操作需要肌肉记忆，建议先在前几百米当练习场。
3.  **优先升级关键能力：** 比如磁铁持续时间、护盾时长或复活能力，这些会大幅提高长距离奔跑的容错率。
4.  **风险与收集的平衡：** 金币很重要，但不值得为了一两枚金币去撞上明显危险的障碍，优先保证持续奔跑。


- **任务完成：** 完成每日任务可获得额外奖励和经验。
### 操作说明

*   **方向与跳跃：** 使用滑动/按键来控制左转、右转、跳跃和下滑（具体视平台与版本）；
*   **倾斜收集：** 支持重力感应的版本中，可以通过倾斜设备来让角色偏向轨道左右侧收集金币；
*   **道具：** 路途中拾取到的加速、磁铁、护盾等道具会自动生效或通过轻触激活。


- **倾斜操作：** 在部分版本中，倾斜设备可更方便地左右收集金币。
### 常见问题解答（FAQ）


- **问：怎样获得更多金币？**  
    **答：** 通过完成任务、使用磁铁等道具、解锁成就，以及（如版本支持）观看广告等方式获取。

- **问：不同角色有什么区别？**  
    **答：** 角色通常拥有不同外观与起始能力/属性，可随着游戏进度逐步解锁。

- **问：如何获得道具？**  
    **答：** 通过跑酷途中拾取、用金币购买，或完成特定任务获得。

- **问：游戏有终点吗？**  
    **答：** 这是无尽跑酷游戏，理论上可以一直跑下去；挑战在于尽可能跑得更远、拿到更高分。
