*Brain Test 2: Tricky Stories（脳トレテスト2：トリッキーストーリー）*は、Unico Studioが開発・公開した、非常にクリエイティブで挑戦的なパズルゲームです。大人気ゲーム*Brain Test: Tricky Puzzles*の続編にあたります。前作が独立したレベル構成だったのに対し、*Brain Test 2*ではストーリー主導型のモデルを導入しており、キャラクターを中心とした一貫性のある物語（チャプター）の中でパズルが展開されます。

このゲームの核心は「常識にとらわれない思考（いわゆる枠の外で考える発想）」にあります。一見シンプルに見えても、予想外のトリックや罠が仕掛けられた数々のパズルを解き明かさなければなりません。従来の論理が通じないことも多く、想像力をフル活用し、画面上の要素を予期せぬ方法で操作して正解を見つける必要があります。

**主な特徴：**

*   **ストーリー主導のレベル構成：** エージェント・スミスの任務、エミリーの農場ライフ、モンスターハンター・ジョーの冒険など、個性豊かなキャラクターたちの物語を軸にゲームが進行します。各チャプターが短い物語となっており、パズルを解くことでストーリーが進んでいきます。
*   **多様なパズルデザイン：** パズルのバリエーションは非常に豊かで、論理的思考、観察力、一般常識（時にはその逆）、そして創造性が試されます。
*   **高度なインタラクティブ操作：** タップ、ドラッグ、マージ（合成）、スマホを振る、傾ける、ピンチアウト（拡大）など、多彩なアクションでゲーム内の要素を操作します。時には、レベルのタイトル文字やUI（ユーザーインターフェース）そのものに正解が隠されていることもあります。
*   **ユーモア溢れるアートスタイル：** 清潔感があり、鮮やかな漫画風のアートスタイルが特徴です。キャラクターやアニメーションにはユーモアが満載で、失敗した時のアニメーションでさえ思わず笑ってしまうことでしょう。
*   **ヒントシステム：**どうしてもレベルがクリアできない時は、ゲーム内の「電球」アイコンを使ってヒントを得たり、レベルそのものをスキップしたりできます（通常は広告視聴が必要です）。

### **2. 遊び方**

このゲームの醍醐味は「自ら考え、発見する喜び」にあるため、全レベルの答えを直接教えてしまうと面白さが半減してしまいます。そこで、ここでは具体的な答えではなく、パズルを解くための【攻略のコツ】や【考え方】を紹介します。

*   **常識を疑う（枠の外で考える）：** これが最も重要なルールです。日常的な論理を問題に当てはめないでください。「絶対に違うだろう」と最初に除外した選択肢が、実は正解であることがよくあります。
*   **画面上のあらゆるものを触る：**
    *   **ドラッグ：** 背景の雲や太陽、あるいは「問題文の文字」まで、動かせそうなものはすべてドラッグしてみてください。
    *   **組み合わせる：** 2つ以上のアイテムを近づけたり重ねたりして、新しい道具が作れないか試しましょう。
    *   **タップ／こする：** あやしい場所を連打したり、指でこすったりすると、隠された仕掛けが動くことがあります。
    *   **拡大／縮小：** 2本の指でピンチ操作をして、隠れたものを探したり、逆に小さくして隙間を通したりできるか試してください。
*   **問題文をよく読む：** 問題文そのものが重要なヒントになっているケースが多々あります。言葉遊びやキーワード、あるいは隠された指示に注意を払ってください。
*   **スマホの機能を活用する：**
    *   **振る：** スマホ本体を振ることで、木から物を落としたり、寝ている人を起こしたりできるレベルが多くあります。
    *   **傾ける：** グラスから水をこぼしたり、ボールを転がしたりするために、スマホを傾ける必要があります。重力が変化します。
    *   **音量ボタン：** まれに、スマホの音量ボタンを押すことがパズルの解法になっている場合があります。
    *   **回転／逆さま：** スマホを逆さまにしたり回転させたりすることでクリアできることもあります。
*   **細部と数字に注目する：** 画面内のすべてのディテールを注意深く観察してください。物の数を正確に数えたり、何かが別のものの後ろに隠れていないか確認したりしましょう。
*   **失敗を恐れない：** 試行錯誤こそがこのゲームの楽しみです。いろいろなアプローチを試すこと自体が、正解への近道です。
*   **ヒントを賢く使う：** どうしても長時間わからなくて進めない場合は、遠慮なく電球のヒントを使いましょう。答えそのものではなく、思考を導く重要な手がかりを教えてくれます。

### **3. 操作ガイド**

*Brain Test 2* の操作は直感的ですが、その操作自体が謎解きの一部になっていることもあります。主な操作方法は以下の通りです。

| アクション | 説明 | 例 |
| :--- | :--- | :--- |
| **タップ／クリック** | オブジェクトを選択する、アクションを起こす、ボタンを押す。 | 電気を消すスイッチを押す、風船を割る。 |
| **ドラッグ＆ドロップ** | オブジェクトを特定の場所へ動かす、キャラクターにアイテムを渡す。 | 鍵を錠前まで運ぶ、食べ物をキャラクターの口元へ持っていく。 |
| **スワイプ** | 障害物をどける、汚れを拭き取る、画面外へ物を追いやる。 | カーテンを開けて後ろのものを探す、邪魔な石を横にずらす。 |
| **ピンチ（拡大・縮小）** | 画面上のオブジェクトを大きくしたり小さくしたりする。 | 地図を拡大してヒントを探す、岩を小さくして穴を通す。 |
| **デバイスを振る** | スマホの加速度センサーを利用する。 | 木を揺すってリンゴを落とす。 |
| **デバイスを傾ける** | ゲーム内の重力の方向を変える。 | スマホを傾けてコップの水をこぼす。 |
| **長押し** | オブジェクトを押し続けて特別な効果を発動する。 | 決定ボタンを長押しして機械を作動させ続ける。 |

### **4. よくある質問 (FAQ)**

**Q:どうしてもレベルがクリアできません。どうすればいいですか？**
**A:** まずは「遊び方」のセクションで紹介した基本的な戦略をすべて試してみてください。画面上のすべての要素に触れ、問題文を深読みし、スマホの傾きや振動機能も試しましょう。それでも手詰まりの場合は、「電球」アイコンをタップしてヒントをもらってください。最終手段として、広告動画を視聴することでそのレベルをスキップすることも可能です。

**Q: このゲームは無料ですか？**
**A:** はい、*Brain Test 2* は基本プレイ無料（F2P）ですが、広告が含まれています。ヒントを得たりレベルをスキップするために広告を見る必要があります。よりスムーズに楽しみたい場合は、アプリ内課金ですべての強制広告を削除することもできます。

**Q: 「電球（ヒント）」はどうすれば増やせますか？**
**A:** ヒントは以下の方法で入手できます。
1.  特定のレベルやチャプターをクリアした報酬として。
2.  毎日のログインボーナスとして。
3.  任意の動画広告を視聴することで（リワード広告）。
4.  アプリ内課金で購入することで。

**Q: Brain Test 1 と Brain Test 2 の違いは何ですか？**
**A:** 最大の違いは「構造」です。前作 *Brain Test 1* は関連性のないパズルが単発で続く形式でしたが、*Brain Test 2* は「ストーリーチャプター」形式を採用しています。各チャプターには決まった主人公と連続したストーリーがあり、パズルを解く目的がより明確で、物語としても楽しめるようになっています。

**Q: インターネット接続は必要ですか？**
**A:** ゲーム自体はオフラインでもプレイ可能です。ただし、ヒントシステムを利用する場合（電球補充のための広告視聴や、レベルスキップ）にはインターネット接続が必要になります。
