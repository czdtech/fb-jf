### ゲーム紹介

Run 3は、Runシリーズの人気エントリーで、無限の宇宙トンネルを通してキャラクターを走らせます。壁にジャンプして重力を変更し、ギャップを避け、できるだけ遠くに走ります。探索モードとエンドレスモード。

### ゲームプレイ戦略と攻略法

1.  **壁を使用:" 重力を変更するために壁にジャンプします。
2.  **先を見る:" 次のギャップを予測します。
3.  **キャラクターをアンロック:" 各キャラクターには独自の能力があります。
4.  **トンネルを探索:" すべてのルートを見つけます。
5.  **練習:" 繰り返しでタイミングが改善します。

### 操作ガイド

- 矢印キー/WASD：移動、ジャンプします。
- スペース：一時停止します。
- トンネルから落ちないでください。
