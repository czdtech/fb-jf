### ゲーム紹介

Run 3は、宇宙トンネルを走り抜けるRunシリーズの作品です。壁にジャンプして重力を切り替え、ギャップを避けながら先へ進みます。探索モードとエンドレスモードが用意されています。

### ゲームプレイ戦略と攻略法

1.  **壁を使う:** 壁にジャンプして重力を切り替えます。
2.  **先を見る:** 次のギャップを予測します。
3.  **キャラクターを使い分ける:** キャラクターごとに挙動が違う場合があります。
4.  **ルートを探索:** 安全な道や近道を探します。
5.  **練習:** 繰り返しでタイミングが安定します。

### 操作ガイド

- 矢印キー/WASD：移動、ジャンプします。
- スペース：一時停止します。
- トンネルから落ちないでください。
