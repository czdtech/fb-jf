### Présentation

Kitchen Sorting te met dans la peau d'un commis qui doit trier en urgence ce qui passe sur le plan de travail : ingrédients, outils, déchets, produits dangereux… Les niveaux ajoutent des éléments trompeurs (cru vs cuit), des objectifs de combo (trier trois articles sains d'affilée) et des pénalités en cas de mauvais placement ou de retard. La difficulté augmente avec des tapis roulants simultanés et des règles d'hygiène plus strictes qui punissent les contaminations croisées.

### Stratégies de tri

1.  **Règles prioritaires** : commence par les splits critiques pour l'hygiène (cru vs prêt à manger) avant les catégories fines.
2.  **Indices visuels** : apprends les pictogrammes (recyclage, thermomètres, etc.) pour gagner en vitesse.
3.  **Tampon** : si le jeu le permet, mets de côté les objets douteux quelques instants plutôt que de faire une erreur.
4.  **Gestion des combos** : enchaîne les objets évidents pour garder les multiplicateurs, et place les items difficiles dans les creux.
5.  **Limiter les erreurs** : une faute coûte plus cher qu'une petite hésitation : mieux vaut rater un combo que cliquer au mauvais endroit.

### Commandes

- Glisse ou tape sur les objets pour les placer dans les bacs correspondants ; pause/retry via l'interface ; les indices peuvent éclairer les bonnes zones.

### Foire aux questions (FAQ)

- **Q : Je confonds recyclables et déchets, un conseil ?**
  R : Mémorise quelques repères fixes (verre, canettes, carton) comme « toujours recyclable » ; les plastiques incertains vont à la poubelle sauf mention contraire.

- **Q : Mes séries de combos s'interrompent sans cesse.**
  R : Place d'abord les objets faciles pour stabiliser ta série, puis glisse les éléments difficiles pendant les accalmies.

- **Q : Les pénalités sont‑elles sévères ?**
  R : Les niveaux avancés punissent lourdement les erreurs d'hygiène ; saute un objet si tu doutes plutôt que de risquer une croix rouge.
