『60 Second Burger Run』は、閉店間際のバーガーショップに厳格な60秒の制限時間内に到達することを目指す、エキサイティングでカラフルなスピードランプラットフォーマーです！美味しいファストフードが大好きな陽気でぽっちゃりしたヒーローを操作し、一連のチャレンジングなステージをできるだけ早くクリアしていきます。このゲームは、反射神経、正確なジャンプ、そしてタイマーがゼロになる前にできるだけ多くのレベルをクリアするためのルート記憶能力をテストします。

### 60 Second Burger Run の遊び方

-  **目標**：合計60秒でできるだけ多くのステージをクリアします。各ステージの最終目標は、バス停まで走り、バスに乗り、次のレベルに進むことです。
-  **操作**：操作はシンプルです。**矢印キー**または **WASD** キーで左右に走り、ジャンプします。
-  **レベルナビゲーション**：反射神経と注意力を使って非常に速く走る必要があります。高くジャンプし、段差を乗り越え、鋭利な物体を飛び越え、火の穴を跳び越えてゴールに到達しなければなりません。
-  **危険と警告**：道には危険がいっぱいです。ヒーローの下で崩れる可能性のある色付きプラットフォームに注意してください。ドクロマークの道路標識も、前方の崖について警告してくれます。
-  **タイマーとリスポーン**：60秒のタイマーは全体のランに対して適用されます。障害物に当たってもまだ時間が残っていれば、*現在のステージ*の最初からリスタートします。60秒が経過すると、ゲーム全体が終了し、最初のステージからやり直しになります。

### 攻略とヒント

-  **記憶が鍵**：タイマーはレベル間でリセットされないため、主な戦略はルートを記憶することです。ゲームは繰り返しによる学習を促しています。失敗しても落ち込まないでください。各試行が次のランのためのレイアウト学習に役立ちます。
-  **スピードと勢い**：継続的な動きが重要です。ジャンプを計画して、一つのアクションから次のアクションへ止まることなく流れるようにしましょう。序盤のレベルで節約した1秒1秒が、後半のより難しいステージのための時間を稼ぎます。
-  **視覚的な手がかりに注意**：ゲームは危険を警告してくれます。崖を示すドクロマークに注意し、見慣れない色のプラットフォームでは崩れる可能性があるので注意してください。新しいアイテムのルールは、ゲーム内の「小さな天使」によって説明されることが多いです。

### よくある質問 (FAQ)

**Q: ゲームに勝つにはどうすればいいですか？**
**A:** このゲームは無限の「タイムアタック」です。目標は最終的なエンディングに到達することではなく、60秒の制限時間内にいくつのレベルをクリアできるかを見ることです。クリアしたレベル数が多いほど、スコアが高くなります。

**Q: スパイクに当たったり落ちたりするとどうなりますか？**
**A:** 障害物に当たると、現在のレベルの最初から即座にリスタートします。ただし、メインの60秒タイマーは停止もリセットもされないため、ミスは非常に高くつきます。

**Q: 全部で何レベルありますか？**
**A:** このゲームは1分間でどこまで行けるかを見るように設計されています。設計されたレベルの数には限りがあるかもしれませんが、チャレンジは特定の最終レベルに到達することではなく、1回のランでできるだけ多くのレベルをクリアすることにあります。
