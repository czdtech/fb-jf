### Présentation du jeu

Tile Guru utilise la formule du plateau à triple match : prends des tuiles dans un plateau limité ; trois du même type s'effacent. Le puzzle est la gestion des piles — évite d'enterrer des sets dépareillés qui bloquent le plateau et forcent les impasses.

<!-- i18n:section:tips -->
<!-- i18n:section:how-to-play -->
### Guide de jeu

1.  **Discipline des paires :** Ne commence pas trop de sets à la fois ; résous les paires rapidement.
2.  **Contrôle de profondeur :** Prends des tuiles accessibles qui n'enterreront pas les matchs nécessaires.
3.  **Anticipation :** Planifie les 2-3 prochaines prises pour éviter le débordement du plateau.
4.  **Annule sagement :** Annule immédiatement après une prise bloquante.
5.  **Bonus :** Garde les mélanges pour les quasi-impasses.

<!-- i18n:section:controls -->
### Commandes

-   Clique sur les tuiles pour les ajouter au plateau ; trois du même type s'effacent ; indices/mélanges via l'interface.

<!-- i18n:section:faq -->
### Foire aux questions (FAQ)

<!-- i18n:faq:id=faq:tile-guru:my-tray-fills-too-fast-e2a2b50d -->
-   **Q : Mon plateau se remplit trop vite.**
    **R :** Limite les sets ouverts et complète les paires avant d'en commencer de nouvelles.

<!-- i18n:faq:id=faq:tile-guru:which-tiles-first-6dd9b763 -->
-   **Q : Quelles tuiles prendre en premier ?**
    **R :** Les tuiles exposées, peu profondes, qui n'enterreront pas leurs partenaires.
