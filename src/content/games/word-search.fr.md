### Game Introduction

"Word Search" is a classic puzzle game where players find hidden words in a grid of letters, searching horizontally, vertically, and diagonally to complete themed word lists. This is a relaxing brain exercise game that helps improve vocabulary and observation skills.

### Gameplay Strategy

1.  **[FR]** **Systematic Scanning**: Scan the grid systematically by rows or columns, don't jump around randomly.
2.  **[FR]** **Multi-directional Thinking**: Remember words can appear in 8 different directions (horizontal, vertical, diagonal).
3.  **[FR]** **Find Long Words First**: Long words are usually easier to spot; finding them first can narrow the search area.
4.  **[FR]** **Marking Strategy**: Mark words immediately after finding them to avoid repeated searches.
5.  **[FR]** **Theme Association**: Use theme hints to predict what types of words might appear.

### How to Play

*   **[FR]** **Select Words**: Use mouse drag or click to select found words.
*   **[FR]** **Multi-directional Selection**: Can select words from any direction (forward/backward, up/down, diagonal).
*   **[FR]** **View List**: Check the word list to see which words still need to be found.
*   **[FR]** **Hint Feature**: Some versions provide hint features to help find difficult words.

### Frequently Asked Questions

1.  **[FR]** **Q: What directions can words be in?**
**A**: Words can appear horizontally (left/right), vertically (up/down), or diagonally, and can be forward or backward.
2.  **[FR]** **Q: How can I improve search efficiency?**
**A**: Use systematic scanning, find long words first, utilize theme clues, and maintain patience and focus.
3.  **[FR]** **Q: What if I can't find the last few words?**
**A**: Carefully check if you've missed certain directions, especially diagonals and backward words.
4.  **[FR]** **Q: Is there a time limit?**
**A**: Most versions have no time limit; you can search at your own pace.
