# GLOSSARY (ja) - 日语术语表

## 1. 游戏类型/机制

- puzzle → パズル
- casual → カジュアル
- idle / clicker → 放置 / クリッカー（统一：**放置系クリッカー**）
- rhythm → リズムゲーム
- horror → ホラー
- platformer → プラットフォーマー / アクション（优先用 **プラットフォーマー**）
- runner → ランナー / エンドレスラン（统一：**ランナー**）
- shooter → シューティング
- strategy → ストラテジー / 戦略（正文优先 **戦略**）

## 2. UI/系统词

- Play / Start → プレイ / 開始（按钮：**プレイ**）
- Pause → 一時停止
- Resume → 再開
- Level / Stage → レベル
- Mode → モード
- Score / High score → スコア / ハイスコア
- Lives → ライフ / 残機（统一：**ライフ**）
- Timer → タイマー
- Power‑up → パワーアップ
- Combo → コンボ
- Leaderboard → ランキング
- Daily challenge → デイリーチャレンジ
- Achievements → 実績
- Upgrade → アップグレード
- Unlock → アンロック
- Fullscreen → 全画面
