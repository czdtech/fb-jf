# GLOSSARY (de) - 德语术语表

## 1. 游戏类型/机制

- puzzle → Puzzle
- casual → Casual
- idle / clicker → Idle / Clicker（保留英文）
- rhythm → Rhythmusspiel
- horror → Horror
- platformer → Plattformer / Plattformspiel（统一：**Plattformspiel**）
- runner → Runner / Endless Runner（保留 Runner）
- shooter → Shooter / Schießspiel（正文统一：**Shooter**）
- strategy → Strategie

- Play / Start → Spielen / Starten（按钮：**Spielen**）

- Resume → Fortsetzen
- Level / Stage → Level
- Mode → Modus
- Score / High score → Punktzahl / Highscore
- Lives → Leben
- Timer → Timer
- Power‑up → Power‑up / Bonus（统一：**Power‑up**）
- Combo → Combo
- Leaderboard → Bestenliste
