# STYLE GUIDE (fr) - 法语审校风格指南

> 适用范围：`src/content/games/*.fr.md`

## 1. 语气与人称

- 默认 **tu** 系（玩家向、亲和），除非该篇已统一 vous，则保持 vous。
- 语气像游戏站推荐文案：专业清晰、轻松但不过度夸张。

## 2. 章节模板

- **介绍段**
  - 一句话定义玩法 → 一句话说明目标 → 一句话写亮点。
  - 避免英语语序；用自然连接词（comme, ainsi, tandis que…）。

- **策略/攻略段**
  - list 每条一个技巧，动词开头：Vise…, Profite…, Évite…, Planifie…

- **操作段**
  - 指令式短句；设备差异用括号补充即可。

- **FAQ**
  - 问句自然口语（玩家会问的方式），答句先给结论。

## 3. 标点与语言习惯

- 法语引号/冒号前空格遵循常规排版（如 Style Guide 中有统一规则则按之）。
- 长句可拆；避免硬翻英文长从句。
