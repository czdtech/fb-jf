# STYLE GUIDE (de) - 德语审校风格指南

> 适用范围：`src/content/games/*.de.md`

## 1. 语气与人称

- 玩家向二人称：默认 **du**，除非全文已使用 Sie。
- 语气直接清晰、信息密度高但易读。

## 2. 章节模板

- **介绍段**
  - 类型/玩法一句 → 目标一句 → 亮点一句。
  - 英文长句优先拆成短句。

- **策略/攻略段**
  - list 动词开头：Ziele…, Nutze…, Vermeide…, Plane…
  - 注意逻辑连接，别硬套英文语序。

- **操作段**
  - 短指令 + 设备说明；避免冗长解释。

- **FAQ**
  - 问句自然德语表达；答句先结论再补充。

## 3. 语言习惯

- 复合名词自然、不要为了直译拼太长。
- 名词大写、动词位置符合德语习惯。
