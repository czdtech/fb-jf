# STYLE GUIDE (ko) - 韩语审校风格指南

> 适用范围：`src/content/games/*.ko.md`

## 1. 语气与人称

- 玩家向说明体：`~하세요/~하면` 或 `~습니다` 均可，但一篇内必须统一。
- 口吻亲和清晰，像本地游戏平台的玩法介绍。

## 2. 章节模板

- **介绍段**
  - 첫 문장: 게임 장르/핵심 플레이  
  - 두 번째: 목표/승리 조건  
  - 세 번째: 특징/재미 포인트

- **전략/팁**
  - 리스트 항목은 동사형으로 시작: `조준하세요`, `벽 반사를 활용하세요`, `다음 버블을 확인하세요` …

- **조작**
  - 간결한 명령문: `마우스로 조준하고 클릭해 발사합니다.`  

- **FAQ**
  - 질문은 자연스러운 구어체, 답은 결론 → 짧은 설명.

## 3. 语言习惯

- 英文专有名保留英文或常用外来语写法；不随意创造译名。
- 术语按 `GLOSSARY.ko.md` 统一。
