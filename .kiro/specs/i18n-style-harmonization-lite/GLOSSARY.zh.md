# GLOSSARY (zh) - 中文术语表

## 1. 游戏类型/机制

- puzzle → 益智 / 解谜（统一：**益智**）
- casual → 休闲
- idle / clicker → 放置 / 点击（统一：**放置点击**）
- rhythm → 节奏游戏
- horror → 恐怖
- platformer → 平台跳跃
- runner → 跑酷 / 无尽跑酷（统一：**无尽跑酷**）
- shooter → 射击
- strategy → 策略
- card game → 卡牌游戏

## 2. UI/系统词

- Play / Start → 开始游戏 / 立即游玩（按钮：**开始游戏**）
- Pause → 暂停
- Resume → 继续
- Level / Stage → 关卡 / 等级（统一：**关卡**）
- Mode → 模式
- Score / High score → 分数 / 最高分
- Lives → 生命 / 生命值（统一：**生命**）
- Timer → 计时器 / 倒计时（统一：**倒计时**）
- Power‑up → 道具 / 增益（统一：**道具**）
- Combo → 连击
- Leaderboard → 排行榜
- Daily challenge → 每日挑战
- Achievements → 成就
- Upgrade → 升级
- Unlock → 解锁
- Fullscreen → 全屏
