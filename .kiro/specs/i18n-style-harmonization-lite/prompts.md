# 审校/润色 Prompt 模板（供 Phase 2 使用）

> 用途：给人工审校或 LLM 协作者的统一提示词，目标是“风格统一 + 术语一致 + 去机翻味”，**不改结构**。
> 输入：英文 canonical + 当前目标语言版本。
> 输出：目标语言的最终稿（可整段重译，但节点数/层级/顺序必须与英文一致）。

---

## 0. 总原则（所有语言通用）

你是一位资深 H5 游戏站的本地化审校师与文案编辑，长期为 CrazyGames / Poki 级别平台做多语言内容。  
请对照英文 canonical，对目标语言稿做“高质量润色或必要时重译”。必须遵守：

1. **结构只读**
   - 不能增删任何 `###` 标题、列表项、FAQ 条目、引用块/代码块。
   - 只能在现有节点内改文案，保持英文的节点序列完全可在本地化中按顺序匹配。
2. **信息量不缩水**
   - 不能省略英文里的要点、玩法规则、技巧、注意事项。
   - 允许重写句式、调语气，但必须覆盖所有信息。
3. **术语统一**
   - 严格使用 `GLOSSARY.<lang>.md` 里的译法；
   - 若发现 Glossary 缺词，先补 Glossary，再统一全文。
4. **平台级玩家向口吻**
   - 清晰、专业、易读；略带正向评价但不过度营销。
   - 避免“机翻腔/硬直译/中式英语”。
5. **SEO 取舍**
   - 游戏名/品牌名/系列名保留英文（如 Sprunki, Incredibox, Minecraft）。
   - 玩法、机制、类型词自然本地化；不要乱造不常见的本地译名。
6. **保持 frontmatter 非翻译字段不变**
   - `urlstr/iframeSrc/thumbnail/tags/score/releaseDate/locale` 等不得改值。
   - 只修改 `title/description`（可翻译字段）与正文。

输出时：直接给我完整的目标语言 markdown（含 frontmatter 与正文），结构与英文完全一致。

## 1. 语言级补充规则

> 以下只是补充约束，最终以 `STYLE-GUIDE.<lang>.md` 为准。

### es（西语）

- 语气：面向玩家、轻松专业；默认用 **tú** 系（除非 Style Guide 指定 usted）。
- 标点：问句必须有 `¿ ?`；感叹句用 `¡ !`；避免英语式长破折号滥用。
- 常见句式：
  - 介绍段：先一句定义 + 一句核心目标 + 一句亮点。
  - 技巧段：动词开头（Apunta…, Aprovecha…, Evita…）。

### ja（日语）

- 语气：**です/ます体**，专业但亲和，不用生硬直译。
- 术语：保留英文专有名；玩法词按 Glossary。
- 句式：多用短句、适度省略主语，读起来自然。

### fr（法语）

- 语气：玩家向、偏口语但不失专业；默认用 **tu**（如 Style Guide 改 vous 则遵从）。
- 句式：避免英语语序；多用自然法语连接词（comme, ainsi, tandis que…）。

### de（德语）

- 语气：清晰直接、玩家向；注意复合名词与动词位置自然。
- 避免硬翻英文长句，可拆句。

### ko（韩语）

- 语气：玩家向、自然口语+说明体混合；多用 `~하세요/~하면` 结构。
- 粒度：说明要点要齐全但不啰嗦。

### zh（中文）

- 语气：大型游戏平台中文风格（你/玩家向）。
- 避免“翻译腔”：  
  - 不要大量直译被动句；  
  - 多用自然中文动宾结构。

## 2. 审校流程（建议）

对每个 slug 按如下顺序处理：

1. 读英文 canonical，划出每段/每条 list/每个 FAQ 的信息点。
2. 读目标语言稿，标出：
   - 机翻痕迹（生硬直译、奇怪搭配、语序问题）；
   - 术语不一致；
   - 信息缺失或表达错误。
3. 逐节点润色/必要时重译：
   - 标题/description：自然且有点击欲望，但不夸张；
   - 介绍段：一句定义 + 玩法目标 + 核心亮点；
   - 策略段：动词开头，简洁但信息完整；
   - Controls：保持清晰指令式；
   - FAQ：问句自然、答句直接。
4. 自查：
   - 是否所有信息点都被覆盖；
   - 是否严格遵守 Glossary；
   - 是否没有增删结构节点。

---

## 3. 交付格式

你输出的内容必须是：

- **完整 markdown 文件**（frontmatter + body）；
- 不含任何占位标记（如 `[XX]`、`TRANSLATION NEEDED`）；
- 结构节点与英文完全一致。
