# STYLE GUIDE (ja) - 日语审校风格指南

> 适用范围：`src/content/games/*.ja.md`

## 1. 语气与人称

- **です/ます体** 为主，礼貌但不敬语过头；整篇保持一致。
- 面向普通玩家：清晰说明 → 适度评价 → 不做过度营销。

## 2. 章节写法模板

- **介绍段**
  - 先一句话定义类型与玩法，再一句写目标/节奏，最后一句写亮点。
  - 示例开头：  
    - “**Bubble Shooter** は、同じ色のバブルを3つ以上そろえて消す定番パズルです。”  

- **策略/攻略段**
  - 每条 list 用短句 + 解释（必要时）：
    - “高い位置を狙う／壁反射を活用する／次のバブルを見て計画する…”

- **操作段**
  - 指令式写法：  
    - “マウスで狙い、クリックで発射します。”  
    - “スペースキーで入れ替え（対応版のみ）。”

- **FAQ**
  - 问句自然口语化；答句先结论再补一句理由。

## 3. 语言习惯

- 不逐词翻译英文名词组合；优先日语自然语序。
- 片假名/汉字平衡，避免难读汉字堆叠。
- 英文专有名保留英文，可加括号解释一次。
