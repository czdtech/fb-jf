# STYLE GUIDE (zh) - 中文审校风格指南

> 适用范围：`src/content/games/*.zh.md`

## 1. 语气与人称

- 面向玩家的二人称“你/玩家”，亲和专业。
- 像大型游戏平台中文文案：清晰、有点推荐感，但不尬吹。

## 2. 章节模板

- **介绍段**
  - 一句定义类型/玩法 → 一句目标/胜利条件 → 一句亮点。
  - 避免“这个游戏提供了…”这种空泛句。

- **策略/攻略段**
  - 每条 list 一个可执行技巧，动词开头更好：  
    - “优先瞄准高位连接点”“善用墙面反弹”“提前规划下一步”……

- **操作段**
  - 指令式短句，设备差异用括号补充。

- **FAQ**
  - 问句自然玩家口吻；答句先结论再补一句解释。

## 3. 禁止机翻腔

- 避免“进行……”“提供……体验”连用；多用自然动宾结构、短句。
- 少用生硬被动句；必要时拆句。

## 4. 术语与专有名

- 游戏名/品牌名保留英文（如 Sprunki / Incredibox）。
- 玩法/系统词按 `docs/i18n/GLOSSARY.zh.md` 统一。
