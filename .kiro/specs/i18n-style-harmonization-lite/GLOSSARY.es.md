# GLOSSARY (es) - 西语术语表

> 原则：同一术语全站只用一个译法；如需例外，请在本表备注。

## 1. 游戏类型/机制

- puzzle → rompecabezas / puzle（统一用 **puzle**）
- casual → casual
- idle / clicker → idle / clicker（保留英文常用写法）
- rhythm → rítmico / de ritmo（统一用 **de ritmo**）
- horror → terror
- platformer → plataformas
- runner → runner / endless runner（保留 runner）
- shooter → shooter / de disparos（统一用 **de disparos**）
- strategy → estrategia
- card game → juego de cartas

## 2. UI/系统词

- Play / Start → Jugar / Empezar（按钮语境用 **Jugar**）
- Pause → Pausa
- Resume → Reanudar
- Level / Stage → nivel
- Mode → modo
- Score / High score → puntuación / mejor puntuación
- Lives → vidas
- Timer → temporizador
- Power‑up → potenciador
- Combo → combo
- Leaderboard → tabla de clasificación
- Daily challenge → desafío diario
- Achievements → logros
- Upgrade → mejora / mejorar（名词用 **mejora**）
- Unlock → desbloquear
- Fullscreen → pantalla completa
