# GLOSSARY (ko) - 韩语术语表

> 适用范围：`src/content/games/*.ko.md`（正文与可翻译字段）

## 1. 游戏类型/机制

- puzzle → 퍼즐
- casual → 캐주얼
- idle / clicker → 방치형 / 클리커（统一：**방치형 클리커**）
- rhythm → 리듬 게임
- horror → 공포
- platformer → 플랫폼 게임
- runner → 러너 / 엔드리스 러너（统一：**러너**）
- shooter → 슈팅 게임
- strategy → 전략

## 2. UI/系统词

- Play / Start → 플레이 / 시작（버튼：**플레이**）
- Pause → 일시정지
- Resume → 재개
- Level / Stage → 레벨
- Mode → 모드
- Score / High score → 점수 / 최고 점수
- Lives → 라이프
- Timer → 타이머
- Power‑up → 파워업 / 아이템（统一：**파워업**）
- Combo → 콤보
- Leaderboard → 랭킹 / 리더보드（统一：**랭킹**）
- Daily challenge → 데일리 챌린지
- Achievements → 업적
- Upgrade → 업그레이드
- Unlock → 잠금 해제
- Fullscreen → 전체 화면
